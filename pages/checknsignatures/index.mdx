# CheckNSignatures

**A library to verify multiple signatures**

CheckNSignatures provides an easy way to verify multiple signatures in a contract. This can be useful for modules that allow multiple signers, such as a multisig, weighted multisig, a voting system, multi-guardian recovery and much more. CheckNSignatures is audited, tested and can be used in production.
