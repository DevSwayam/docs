# getAccountAutomations

Get all account automations.

## Usage

```typescript copy
const accountAutomations = await automationClient.getAccountAutomations(
  "0x_Account_Address"
);
```

## Parameters

### account

- Type: [`Address`](https://viem.sh/docs/glossary/types#address)

Account address to get automations for.

## Returns

Type [`Automation[]`](/automations/glossary/types#automation)

Array of active automations.
