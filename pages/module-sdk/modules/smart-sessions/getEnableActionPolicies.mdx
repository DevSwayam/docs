# getEnableActionPolicies

Get the action to enable action policies for a specific session in the smart sessions validator.

## Usage

```typescript copy
const action = await getEnableActionPolicies({
  permissionId: '0xPermissionId',
  actionPolicies: [
    {
      actionTarget: '0xTargetAddress',
      actionTargetSelector: '0xTargetSelector',
      actionPolicies: [
        {
          policy: '0xPolicyAddress',
          initData: '0xPolicyInitData',
        },
      ],
    },
  ],
});
```

## Parameters

### permissionId

- Type: [`Hex`](https://viem.sh/docs/glossary/types#hex)

The permission ID of the session to enable action policies.

### actionPolicies

- Type: [`ActionData[]`](/module-sdk/glossary/types#actiondata)

A list of action policies.

## Returns

### action

- Type: [`Promise<Action>`](/module-sdk/glossary/types#action)

The action to enable action policies for a specific session in the smart sessions validator.
