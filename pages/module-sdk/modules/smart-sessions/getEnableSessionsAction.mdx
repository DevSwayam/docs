# getEnableSessionsAction

Get the action to enable sessions in smart sessions validator.

## Usage

```typescript copy
const action = await getEnableSessionsAction({
  sessions: [
      {
        sessionValidator: '0xValiatorAddress',
        sessionValidatorInitData: '0xValiatorInitData',
        salt: toHex(toBytes('1', { size: 32 })),
        userOpPolicies: [],
        erc7739Policies: {
          allowedERC7739Content: [],
          erc1271Policies: [],
        },
        actions: [
          {
            actionTarget: '0xTargetAddress',
            actionTargetSelector: '0x00000000',
            actionPolicies: [
              {
                policy: '0xPolicyAddress',
                initData: '0xPolicyInitData',
              },
            ],
          },
        ],
      },
    ],
});
```

## Parameters

### sessions

- Type: [`Session[]`](/module-sdk/glossary/types#session)

List of sessions to enable in the smart sessions validator.

## Returns

### action

- Type: [`Promise<Action>`](/module-sdk/glossary/types#action)

The action to enable sessions in the smart sessions validator.
