# uninstallModule

Get the action to uninstall a module from an account.

## Usage

```typescript copy
const action = await uninstallModule({
  client,
  account,
  module,
})
```

## Parameters

### client

- Type: `PublicClient`

The rpc client to use for the chain you want to uninstall the module from.

### account

- Type: [`Account`](/module-sdk/glossary/types#account)

The account object.

### module

- Type: [`Module`](/module-sdk/glossary/types#module)

The module object.

## Returns

### action

- Type: [`Promise<Action[]>`](/module-sdk/glossary/types#action)

The action to uninstall the module from the account.
