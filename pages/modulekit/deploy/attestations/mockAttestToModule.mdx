# mockAttestToModule

Make a mock attestation to a module.

## Usage

```solidity copy
mockAttestToModule({ module: module, attestationData: attestationData, moduleTypes: moduleTypes });
```

## Parameters

### module

- Type: `address`

The address of the module to attest.

### attestationData

- Type: `bytes`

The attestation data.

### moduleTypes

- Type: `ModuleType[]`

The module types.

## Returns

None
