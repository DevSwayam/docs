# Tooling Overview

Rhinestone's tooling is designed to make it easy for developers to build with modular smart accounts. Our tools make it easy for developers to build, test, audit, deploy and use modules at the application layer. The tooling is open source and designed to grow the ecosystem of smart accounts. Here is an overview of all our tools:

- [**ModuleSDK**](/module-sdk): A TypeScript library for using smart account modules in applications
- [**ModuleKit**](/modulekit): A development kit for building smart account modules
- [**AutomationsSDK**](/automations): A TypeScript library for using Rhinestone Automations
- [**Safe7579**](/safe7579): An ERC-7579 adapter for Safe accounts
- [**Module Registry**](/module-registry): An attestation registry for securely using smart account modules
- [**SentinelList**](/erc4337-data-libs): A library for linked lists
- [**FlatBytes**](/erc4337-data-libs): A library to store bytes in consecutive storage slots
- [**CheckNSignatures**](/checknsignatures): A library to verify multiple signatures
- [**ERC4337Valiation**](/erc4337-validation): A library to validate the ERC-4337 rules within Foundry
- [**ModuleSandbox**](/sandbox): A frontend playground building and testing smart account modules
